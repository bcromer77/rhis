# RHIS – Regulatory Horizon Intelligence System

RHIS is a foresight engine designed to help researchers, analysts, and innovators surface signals and strategic patterns from global policy discourse.

---

## 🧠 Overview

RHIS enables users to:

- Explore geopolitical and economic narratives across regions
- Search across official transcripts and structured discussions
- Detect subtle shifts in tone, risk posture, and thematic frequency

Built using modern full-stack tools, RHIS is optimized for adaptability across sectors, especially where regulatory clarity, environmental foresight, or social impact is critical.

---

## 🛠️ Tech Stack

- **Frontend:** Next.js + Tailwind CSS
- **Backend:** Node.js + Express
- **Search & Data:** MongoDB + Vector Similarity
- **AI Tools:** Embedding & NLP integrations
